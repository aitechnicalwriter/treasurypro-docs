Learn how to access, filter, and export a log of all system activity and user actions for internal auditing and compliance reporting.

## Step 1: Access the Audit Trail
1. Navigate to **Administration** > **Security** > **Audit Logs**.

## Step 2: Define the Filter Scope
1. Use the **Date Range** filter to select the quarter you need to review.
2. In the **Event Type** filter, select **Payment Approval** to focus on sensitive transactions.
3. In the **User Filter**, select the name of the user whose activity you wish to review.

## Step 3: Export the Data
1. Click the **Export** button in the top right corner.
2. Select **CSV** as the format for easy integration into your spreadsheet software.
3. The report will be generated and delivered to your secure inbox within 5 minutes.
