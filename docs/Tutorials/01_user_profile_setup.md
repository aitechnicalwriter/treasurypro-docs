# Setting Up Your TreasuryPro User Profile

This tutorial guides first-time users through the necessary steps to configure their security settings, set notification preferences, and complete their profile for optimal use of TreasuryPro.

## Step 1: Initialize Your Profile Security

1.  Navigate to **Settings** > **User Profile**.
2.  Set your initial **Two-Factor Authentication (2FA)** device (e.g., Google Authenticator). *This is required for all payment submissions.*
3.  Set a **Security Question** used for account recovery.

## Step 2: Configure Notification Preferences

1.  Select the **Notifications** tab within the User Profile.
2.  Enable **Email Alerts** for the following high-priority events:
    * **Intraday Balance Dropped Below Threshold**
    * **Payment Submission Failed (Error 400 or 500)**
3.  Set your primary **Alert Delivery Timezone** (must match your local time).

## Step 3: Customize Your Dashboard View

1.  Navigate back to the main **Cash Overview Dashboard**.
2.  Click the **Customize** button in the top right corner.
3.  Drag the **Intraday Liquidity Position** widget to the top of your screen for immediate visibility.
4.  Save your layout as **"Default Layout."**

You have now completed the initial setup! You are ready to perform daily tasks like checking balances and submitting payments.
