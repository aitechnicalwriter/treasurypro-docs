# Setting Up a Threshold Alert

Set up automated alerts to notify you when your account balance falls below a critical threshold, enabling rapid liquidity intervention.

## Step 1: Navigate to Alerts Configuration
1. Go to **Settings** > **Notifications** > **Account Alerts**.
2. Click **+ New Threshold Alert**.

## Step 2: Define the Trigger
1. **Select Account:** Choose the primary operating account linked to QBank Connect.
2. **Threshold Value:** Enter the minimum balance required (e.g., $100,000.00).
3. **Trigger Type:** Select **Available Balance** (this uses real-time data).

## Step 3: Configure Notification Channels
1. **Primary Channel:** Check the **Email** box.
2. **Secondary Channel:** Check the **SMS** box (requires mobile number verification).
3. Click **Activate Alert**. The system will now monitor the real-time feed from QBank Connect and notify you immediately if the balance drops below $100,000.
