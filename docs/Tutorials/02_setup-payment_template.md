# Setting Up a New Payment Template

This tutorial shows how to create a pre-filled payment template to expedite routine transactions and ensure compliance with formatting rules.

## Step 1: Access the Template Manager
1. From the main navigation, select **Payments** > **Template Manager**.
2. Click the **+ New Template** button.

## Step 2: Define Template Parameters
1. **Name the Template:** Use a descriptive name (e.g., `MonthlyRent_VendorID401`).
2. **Select Payment Type:** Choose **Domestic ACH Transfer**.
3. **Set Default Account:** Select the QBank checking account that will typically originate this payment.

## Step 3: Add Beneficiary and Amount Rules
1. Enter the required **Beneficiary Bank Details** and **Account Number**.
2. **Optional:** Set a **Minimum** and **Maximum** allowed transaction amount for this template to prevent high-value input errors.

## Step 4: Save and Share
1. Click **Save Template**.
2. To allow colleagues to use this template, click the **Share Permissions** icon and assign the **Payment Creator** role to the relevant team members.
